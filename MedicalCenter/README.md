# Online Doctor Appointment Platform

An online doctor appointment web application that makes it easy for patients to find doctors and book appointments quickly. Instead of visiting hospitals just to check doctor availability, users can browse doctor profiles, view their specialties, and schedule appointments in a few clicks.

The platform helps patients find the right healthcare professional based on their needs, including specialties such as **General Physician, Gynecologist, Dermatologist, Pediatrician, and Neurologist**. The goal of this project is to simplify the appointment process, save time, and provide a smooth experience for both patients and clinics.

## Tech Stack

* **React** – For building reusable UI components
* **Vite** – Fast development and build tool
* **Tailwind CSS** – Responsive and modern styling
* **React Router** – Seamless page navigation without reloads
